Triage: Hospital ER Android Application

Android Developer

This was a team-based course project for one of my second year software development classes. Our team created an Android application which stored various information about ER patients, such as vital signs, symptoms and personal credentials.

Using this information, the application performed an algorithm which ranked patients according to how urgently they needed to be seen by a doctor. In practice, the application would allow doctors to tend to patients in a more efficient and quicker manner.

Java, XML, Android, Software Design, Mobile Development

Pigeon Passage Android Game

Android Developer

I created Pigeon Passage, with the help of a talented pixel art designer, as part of a summer project supervised by University of Toronto Computer Science Lecturer, Steve Engels. It was created using the GameMaker Engine. Pigeon Passage was created to promote the Royal Ontario Museum's 100th anniversary additions to the biodiversity exhibit. The game went from idea to live presentations within the course of a few weeks and is often displayed for public use at various promotional events at the museum.

Scripting, GameMaker Engine, Android, Game Development
